# Bản thử hướng mới

Đây là bản sao của dự án CVMS hiện tại, dùng để thử hướng phát triển khác mà không ảnh hưởng bản chính.

- Bản chính: `cvms-dacv-hoan-chinh-ai`
- Bản thử: `cvms-dacv-huong-moi`
- Port bản chính: `4182`
- Port bản thử: `4183`

Chạy bản thử:

```powershell
cd C:\Users\phanh\Downloads\CVMS_DACV_HOAN_CHINH_AI\cvms-dacv-huong-moi
npm install
npm run dev
```

Sau đó mở:

```text
http://localhost:4183/login.html
```

Lưu ý: bản thử đã copy code và dữ liệu hiện tại để bạn có thể thử nghiệm riêng. Khi sửa bản thử, bản chính không bị ảnh hưởng.
